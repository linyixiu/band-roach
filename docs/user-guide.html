安裝到主畫面

iPhone／iPad

  1. 使用 Safari 開啟 Band Roach。
  2. 點下方「分享」按鈕。
  3. 選擇「加入主畫面」，再按「加入」。

Android

  1. 使用 Chrome 開啟 Band Roach。
  2. 點右上角選單，選擇「安裝應用程式」或「加到主畫面」。
  3. 按「安裝」。

電腦版 Chrome／Edge

點網址列右側的安裝圖示；如果畫面出現「＋ 安裝 Band Roach」，也可以直接使用。

三步完成級數換算

1選擇調性

例如 C、F、B♭，再選大調或自然小調。

2輸入級數

輸入 1645，畫面會立刻顯示和弦組合。

3點擊和弦

可以聽聲音，並在下方查看建議吉他指法。

例：C 大調輸入 1645，會得到 C → Am → F → G。

級數與和弦怎麼輸入？

1645連續級數

最常用的快速輸入方式。

1 6 4 5空格分隔

結果與 1645 相同。

b3 #4升降級數

支援 b／♭ 與 #／♯。

5(7)屬七和弦

括號表示延伸，不會被當成「5、7」兩個和弦。

1maj7大七和弦

也支援 m7、9、sus2、sus4、add9、6、m9、11、13 等。

5(9)/2Slash Chord

斜線後面表示指定低音。

使用 Capo

  1. 先選調並輸入級數。
  2. 拖曳「Capo」滑桿到實際夾的位置。
  3. 上方和弦卡會改成「你的手要按」的和弦；原本實際發聲的和弦仍會保留供對照。

如果畫面出現「初學者建議」，可以點「套用建議」，用較多開放和弦取代難按的封閉和弦。

查看吉他指法

  1. 點上方任一和弦卡。
  2. 往下查看六弦圖與琴格位置。
  3. 若有多種按法，可切換「標準／可靠移調／其他把位／好切換」。
  • ×：這條弦不彈。
  • ○：空弦。
  • 圓點內 1–4：建議使用的手指，1 是食指、4 是小指。
系統找不到可靠按法時會直接說明，不會拿另一個和弦假裝代替。

播放和弦

可選木吉他、電吉他或柔和鋼琴。點單一和弦卡可聽該和弦，按「播放」會依序播放整組。

手機與電腦瀏覽器通常要求使用者先點擊，才允許播放聲音；第一次沒有聲音時，再按一次「播放」即可。

拍照轉譜

  1. 選擇「直接拍照」或「選擇照片」；電腦也可把圖片拖進區塊。
  2. 按「辨識級數」，等待辨識完成。
  3. 按「框選」，在圖片上按住並上下拖曳，框住譜面上方的和弦級數列。框選會自動橫跨整張圖片。
  4. 和弦出現後，點選標記即可修改級數、和弦名稱、微調位置或刪除。
  5. 需要補充時按「＋ 手動新增」。完成後可匯出 PNG 或 PDF。
辨識建議：照片盡量平整、清楚、光線均勻。辨識仍可能出錯，練團前請快速核對和弦。

自動保存與隱私

調性、級數、Capo、照片、框選與手動修正會自動保存在目前裝置，重開 App 可繼續上次進度。

轉譜圖片不會上傳到 Band Roach 伺服器。若要刪除本機紀錄，請在轉譜區按「清除目前樂譜」。

常見問題

看不到最新版

完全關閉已安裝的 App 再開一次；瀏覽器版也可以重新整理頁面。

沒有聲音

確認裝置音量與靜音模式，然後再次按播放。電腦瀏覽器也可檢查分頁是否被靜音。

拍照辨識失敗

確認網路連線，改用更清楚、角度更正的圖片；也可以使用「＋ 手動新增」。

和弦或指法看起來不合理

回報時請附上調性、輸入的級數、Capo 格數與畫面截圖，會更容易重現問題。
